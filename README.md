# AI Readmission Risk Copilot

Full-stack AI healthcare application using IBM watsonx to predict hospital readmission risk and generate explainable insights.

## Tech Stack
- FastAPI (backend)
- React (frontend)
- IBM watsonx (AutoAI + Prompt Lab)
- Docker + IBM Cloud Code Engine

## Features
- Predict readmission risk from patient data
- Generate AI-powered explanations
- Structured JSON outputs for clinical insights
